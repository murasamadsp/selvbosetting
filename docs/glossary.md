# Глосарій: Норвезькі терміни

Усі ключові слова, які зустрічаються в процесі Avtalt Selvbosetting.

______________________________________________________________________

## Організації та люди

| Термін | Що це |
| ------ | ------ |
| **IMDi** | Integrerings- og mangfoldsdirektoratet — державний орган, що координує розселення біженців по комунах. Укладає угоди з комунами. |
| **UDI** | Utlendingsdirektoratet — міграційна служба. Видає рішення (vedtak) про надання захисту. |
| **NAV** | Arbeids- og velferdsetaten — служба зайнятості та соціального захисту. Видає допомогу, гарантії депозиту. |
| **KVS** | Kvalifiseringssenteret — відділ комуни, що відповідає за інтеграцію та розселення біженців (назва може відрізнятися в кожній комуні). |
| **Flyktningtjenesten** | Служба у справах біженців при комуні або NAV. |
| **Integreringskonsulent** | Ваш особистий куратор у комуні після схвалення переїзду. |

______________________________________________________________________

## Документи та системи

| Термін | Що це |
| ------ | ------ |
| **UDI Vedtak** | Офіційне рішення UDI про надання вам тимчасового захисту. Багатосторінковий PDF — надходить у Digipost або Altinn. |
| **Oppholdskort** | Пластикова картка-посвідка на проживання (ПМП/ВНЖ). Приходить поштою від поліції. |
| **DUF-nummer** | Ваш унікальний номер у системі UDI. Вказаний на oppholdskort і в vedtak. |
| **Digipost** | Норвезька цифрова поштова скринька. Туди надходять офіційні листи від UDI, NAV, Skatteetaten. |
| **Altinn** | Державний портал для електронного документообігу. Альтернативне місце отримання vedtak. |
| **Folkeregisteret** | Державний реєстр населення. Реєстрація нової адреси — через Skatteetaten. |

## Процес розселення

| Термін | Що це |
| ------ | ------ |
| **Avtalt selvbosetting** | Розселення за погодженням: ви самі знайшли квартиру, але переїзд офіційно узгоджений між комуною та IMDi. Зберігає всі права. → [Покроковий Гайд](guide/index.md) |
| **Selvetablering** | Самовільне розселення без погодження з комуною. Призводить до втрати права на introduksjonsprogram та виплати. |
| **Bosetting** | Офіційне розселення (реєстрація в комуні). |
| **Vedtak** | Офіційне рішення органу влади (UDI, NAV, комуни). Будь-яке рішення — завжди vedtak. |
| **Usignert kontrakt** | Непідписаний чорновик договору оренди — потрібен для подачі в комуну до схвалення. |
| **Anmodning** | Запит IMDi до комуни прийняти певну кількість біженців на рік. |

## Гроші та виплати

| Термін | Що це |
| ------ | ------ |
| **Introduksjonsstønad** | Щомісячна виплата учасникам інтеграційної програми. Розраховується як частка від 2G. → [Виплати та Гарантії](finances/payments.md) |
| **Introduksjonsprogram** | Програма інтеграції: мовні курси + практика, 30–37.5 год/тиждень. За участь отримуєте introduksjonsstønad. → [Докладніше](finances/program.md) |
| **G (Grunnbeløp)** | Базова одиниця розрахунку виплат. Оновлюється щороку 1 травня. 1G = 130 160 NOK (з травня 2025). |
| **Depositumsgaranti** | Гарантійний лист від NAV замість готівки на депозит. Юридичне зобов'язання держави перед орендодавцем. → [Як саме платять](finances/how-payments-work.md) |
| **Direkte utbetaling** | Пряма виплата оренди від NAV напряму на рахунок орендодавця. → [Як саме платять](finances/how-payments-work.md) |
| **Sosialhjelp / Sosialtjeneste** | Соціальна допомога від NAV. Покриває базові витрати (їжа, оренда) до старту introduksjonsprogram. |
| **Nødhjelp** | Екстрена допомога від NAV — готівка або передоплачена картка. Якщо немає банківського рахунку або грошей. → [Банківський рахунок](finances/bank-account.md) |
| **Etableringstilskudd** | Разова виплата від NAV на облаштування квартири (меблі, техніка). |
| **Bostøtte** | Житлова субсидія від Husbanken для людей з низьким доходом і високою орендою. → [Виплати та Гарантії](finances/payments.md) |
| **Integreringstilskudd** | Грант від IMDi, який комуна отримує за кожну прийняту людину. Стимул для комуни приймати біженців. |

## Оренда та житло

| Термін | Що це |
| ------ | ------ |
| **Husleiekontrakt** | Договір оренди. Стандартний бланк — від Forbrukerrådet. → [Шаблони листів](templates/landlord.md) |
| **Depositum** | Заставна сума (зазвичай 3 місяці оренди). Максимум за законом — 6 місяців. |
| **Oppsigelse / Oppsigelsestid** | Розірвання договору / термін повідомлення про виїзд (стандарт — 3 місяці). |
| **Overtakelsesprotokoll** | Акт приймання квартири. Підписується при заїзді — фіксує стан житла. → [Чек-лист приймання](checklists/move-in.md) |
| **Nøktern bolig** | «Скромне житло» — критерій комуни: вартість оренди має відповідати майбутньому доходу після закінчення програми. |
| **Godkjent for utleie** | Офіційний статус квартири як дозволеної для здачі в оренду. |
| **Leveringsplikt** | Дефолтний тариф електроенергії — вмикається автоматично, якщо немає контракту. Значно дорожчий за ринок. |
| **Husordensregler** | Правила будинку (тиша, сортування сміття тощо). |
