# Листи орендодавцю

Три готові шаблони для різних ситуацій. Напишіть **одним повідомленням** — у чаті Finn.no або на e-mail орендодавця.

## Коли який варіант

| Варіант | Коли використовувати |
| ------- | -------------------- |
| **Варіант 1: Короткий** | Перший контакт, невідомо чи приймають NAV-гарантію |
| **Варіант 2: Якщо будете сусідами** | Господар живе в тому ж будинку (hybel, sokkel) |
| **Варіант 3: Конкретний** | Оголошення вже вказує «NAV-garanti OK» і є конкретна адреса |

!!! tip "Персоналізуйте"

    Замініть «22 роки, звукорежисер» на ваш вік і фах. Конкретна людина — більше довіри.
