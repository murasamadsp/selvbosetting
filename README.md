# Avtalt Selvbosetting: Посібник (2026)

Відкрита база знань для процесу **«Avtalt Selvbosetting»** (Розселення за погодженням) у Норвегії.

Допомагає легально переїхати з транзитного центру в орендовану квартиру в бажаній комуні, зберігши виплати та право на інтеграційну програму.

## Структура

```
docs/
├── index.md                    # Головна
├── start.md                    # З чого почати
├── guide/index.md              # Покроковий гайд
├── laws/index.md               # Закони та права
├── templates/                  # Шаблони листів
├── communes/                   # Досьє: Осло, Ліллестрьом, Нордре Фолло
├── finances/                   # Виплати, NAV-garanti, механізм оплати
├── checklists/                 # Виїзд із муттака, перевірка квартири
├── daily.md                    # Побут та облаштування
└── faq/                        # FAQ та якщо відмовили
```
